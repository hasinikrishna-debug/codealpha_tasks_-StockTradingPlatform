# Stock Trading Platform

## Overview

A Java-based Stock Trading Platform that simulates stock market operations. Users can buy and sell stocks, manage portfolios, track transactions, and save portfolio data using file handling.

## Features

* Display stock market data
* Buy and sell stocks
* Portfolio management
* Transaction history
* File-based portfolio storage
* Object-Oriented Programming (OOP)

## Technologies Used

* Java
* HashMap
* ArrayList
* File Handling
* OOP Concepts

## How to Run

Compile:
javac *.java

Run:
java Main
